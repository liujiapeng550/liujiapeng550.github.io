## 动态库和静态库

动态库：运行时加载，编译中只是引用，可执行文件小 (.so, .dll,lib) 

静态库：编译的时候使用加载到可执行文件，可执行文件大(.a,.lib)



## 编译流程图

|            | Unix | windows |
| ---------- | ---- | ------- |
| 目标模块   | *.o  | **.obj  |
| 静态函数库 | *.a  | **.lib  |
| 动态函数库 | *.so | **.dll  |
| 程序       | +x   | **.exe  |
